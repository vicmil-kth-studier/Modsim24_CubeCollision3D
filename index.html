2 Mars 2024: The project idea

The basic idea of the project is to implement collisions between cubes in 3D. I would also like to make much of the code reusable for later projects, so I will try to make a library of sorts of help functions and classes.

I am planning to write the project in C++ and use OpenGL coupled with SDL for the graphics part. I will also rely on a library called GLM for some of the linear algebra and maths involved in the project.

3 Mars 2024: Running C++ code in the web

I thought it would be really cool to be able to interact with my project in the web. That way people don't have to download anything. I chose to use C++ for my project, and it turns out that there exists a compiler called emscripten that allows you to compile your code into webassemply that can then be run in the web.

I created some simple hello world program and compiled it. The result can be found in the link below:
